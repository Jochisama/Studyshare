🟢 Exemple 1 — Le plus simple possible

Objectif : comprendre ce que fait la convolution étape par étape.
x(n) — signal d'entrée (Nₓ = 3)
1n=0
2n=1
3n=2
h(n) — réponse impulsionnelle (Nₕ = 2)
1n=0
2n=1
Taille de y(n) : Nᵧ = Nₓ + Nₕ − 1 = 3 + 2 − 1 = 4
y(n) = Σ x(m) · h(n − m)

📋 Calcul pas à pas

0
Calcul de y(0)
y(0) = x(0)·h(0) = 1×1 = 1
On ne prend que m=0 (les autres termes ont h(0-m) hors domaine)
1
Calcul de y(1)
y(1) = x(0)·h(1) + x(1)·h(0)
= 1×2 + 2×1 = 2 + 2 = 4
2
Calcul de y(2)
y(2) = x(1)·h(1) + x(2)·h(0)
= 2×2 + 3×1 = 4 + 3 = 7
3
Calcul de y(3)
y(3) = x(2)·h(1) = 3×2 = 6

✅ Résultat final

1y(0)
4y(1)
7y(2)
6y(3)
💡 Intuition : la convolution c'est comme "glisser" h retourné sur x et multiplier les valeurs qui se superposent.

🟡 Exemple 2 — Niveau intermédiaire

Nouveauté : séquences un peu plus longues, avec des valeurs négatives.
x(n) — (Nₓ = 4)
2n=0
-1n=1
3n=2
1n=3
h(n) — (Nₕ = 3)
1n=0
0n=1
-1n=2
Taille de y(n) : Nᵧ = 4 + 3 − 1 = 6
y(n) = Σ x(m) · h(n − m)

📋 Calcul pas à pas

0
y(0)
= x(0)·h(0) = 2×1 = 2
1
y(1)
= x(0)·h(1) + x(1)·h(0) = 2×0 + (−1)×1 = 0 − 1 = −1
2
y(2)
= x(0)·h(2) + x(1)·h(1) + x(2)·h(0)
= 2×(−1) + (−1)×0 + 3×1 = −2 + 0 + 3 = 1
3
y(3)
= x(1)·h(2) + x(2)·h(1) + x(3)·h(0)
= (−1)×(−1) + 3×0 + 1×1 = 1 + 0 + 1 = 2
4
y(4)
= x(2)·h(2) + x(3)·h(1) = 3×(−1) + 1×0 = −3
5
y(5)
= x(3)·h(2) = 1×(−1) = −1

✅ Résultat final

2y(0)
-1y(1)
1y(2)
2y(3)
-3y(4)
-1y(5)

🔴 Exemple 3 — Celui du cours

C'est exactement l'exemple vu dans le cours ! On va le détailler encore plus.
x(n) — (Nₓ = 5, commence à n=−1)
-1n=-1
1n=0
2n=1
3n=2
3n=3
h(n) — (Nₕ = 3, commence à n=0)
1n=0
-1n=1
1n=2
Taille de y(n) : Nᵧ = 5 + 3 − 1 = 7 — y(n) va de n=−1 à n=5

📋 Calcul détaillé

−1
y(−1)
= x(−1)·h(0) = (−1)×1 = −1
0
y(0)
= x(−1)·h(1) + x(0)·h(0) = (−1)×(−1) + 1×1 = 1 + 1 = 2
1
y(1)
= x(−1)·h(2) + x(0)·h(1) + x(1)·h(0)
= (−1)×1 + 1×(−1) + 2×1 = −1 −1 + 2 = 0
2
y(2)
= x(0)·h(2) + x(1)·h(1) + x(2)·h(0)
= 1×1 + 2×(−1) + 3×1 = 1 − 2 + 3 = 2
3
y(3)
= x(1)·h(2) + x(2)·h(1) + x(3)·h(0)
= 2×1 + 3×(−1) + 3×1 = 2 − 3 + 3 = 2
4
y(4)
= x(2)·h(2) + x(3)·h(1) = 3×1 + 3×(−1) = 3 − 3 = 0
5
y(5)
= x(3)·h(2) = 3×1 = 3

✅ Résultat final

-1y(-1)
2y(0)
0y(1)
2y(2)
2y(3)
0y(4)
3y(5)
✔️ Même résultat que dans le cours : {−1, 2, 0, 2, 2, 0, 3}

⚡ Méthode Rapide — Tableau de multiplication

Astuce : comme une multiplication à la main ! On multiplie chaque coefficient de h par toute la séquence x, puis on décale et on additionne.

Reprise de l'Exemple 1 : x = {1, 2, 3} et h = {1, 2}

Opération pos 0 pos 1 pos 2 pos 3
x × h(0)=1 1 2 3 0
x × h(1)=2 (décalé d'1) 0 2 4 6
y(n) = somme 1 4 7 6

Exemple 2 : x = {2, −1, 3, 1} et h = {1, 0, −1}

Opération pos 0 pos 1 pos 2 pos 3 pos 4 pos 5
x × h(0)=1 2 -1 3 1 0 0
x × h(1)=0 (décalé) 0 0 0 0 0 0
x × h(2)=−1 (décalé ×2) 0 0 -2 1 -3 -1
y(n) = somme 2 -1 1 2 -3 -1
Règle du décalage :
La ligne correspondant à h(k) est décalée de k positions vers la droite avant de sommer.

📐 Résumé de la méthode en 3 étapes

1
Écrire x(n) en ligne
Pour chaque coefficient b de h, multiplier toute la séquence x par b
2
Décaler chaque ligne
La ligne du k-ième coefficient de h est décalée de k colonnes
3
Sommer les colonnes
La somme de chaque colonne donne y(0), y(1), y(2)...
