CH 1

Reconnaissance de Formes — Fondements

Définition RF : associer une étiquette (classe) à une donnée. Des données différentes peuvent avoir la même étiquette.
3 Approches :
① Statistique : forme = vecteur ℝⁿ → Bayes, KNN, LDA, ACP
② Syntaxique/Structurelle : primitives + grammaires → formes complexes
③ Connexionniste : réseaux de neurones → MLP, CNN, RNN
Primitive = composante non décomposable (segment, boucle, arc)
Caractéristique = mesure scalaire (hauteur/largeur, énergie, niveaux de gris)

CH 2

Classifieurs — KNN · Bayes · ACP vs LDA · Confusion Matrix

K-NN

  • Calculer D(X, Pᵢ) pour tous les prototypes
  • Sélectionner les K plus proches
  • Vote majoritaire → classe de X
Euclidienne : √Σ(xₖ-rₖ)² City Block : Σ|xₖ-rₖ|
K=1 → frontière complexe, overfit
K grand → frontière lisse, underfit

Bayes

P(wᵢ/x) = P(x/wᵢ)·P(wᵢ) / P(x) Décide wᵢ si P(x/wᵢ)·P(wᵢ) = MAX P(x) = Σⱼ P(x/wⱼ)·P(wⱼ)
Astuce : P(x) identique pour toutes les classes → comparer directement P(x/wᵢ)·P(wᵢ) sans calculer P(x)

CH 3

SVM — Support Vector Machines

Principe : hyperplan optimal qui maximise la marge 2/‖w‖ entre 2 classes. Seuls les Vecteurs de Support (points sur H₁/H₂) définissent l'hyperplan.
H : ⟨w,x⟩ + b = 0 (frontière) H₁ : ⟨w,x⟩ + b = +1 (classe +1) H₂ : ⟨w,x⟩ + b = -1 (classe -1) Marge = 2/‖w‖ Objectif : min ½‖w‖² s.c. yᵢ(⟨w,xᵢ⟩+b) ≥ 1 Soft margin (non-séparable) : min ½‖w‖² + C·Σεᵢ
NoyauFormuleUsage
Linéaire⟨x,y⟩Données séparables
RBF ★exp(−‖x−y‖²/2σ²)Le + utilisé
Poly(⟨x,y⟩+1)ᵖRelations polynomiales
Tanhtanh(A+B⟨x,y⟩)Type neuronal
C petit → marge large, tolérant → underfit
C grand → marge étroite → overfit
OvO : N(N-1)/2 classif. (vote)
OvA : N classif. (max décision)
CH 4

MLP — Réseaux de Neurones & Rétropropagation

Architecture : Input → Couches cachées (ReLU) → Sortie (Sigmoïde/Softmax)
Sans activation non-linéaire = aucun intérêt d'empiler des couches.
ActivationPlageQuand l'utiliserProblème
Sigmoïde(0,1)Sortie binaireVanishing gradient
Tanh(−1,1)Couches cachéesVanishing gradient
ReLU ★[0,+∞)Couches cachéesDying ReLU
Leaky ReLU(−∞,+∞)Résout dying ReLUα à tuner
Softmax(0,1) Σ=1Sortie multi-classesSortie seulement
Dying ReLU : x<0 → gradient=0 → neurone mort. Fix : Leaky ReLU = max(αx, x).

🔄 Rétropropagation

eⱼ(n) = dⱼ(n) − yⱼ(n) δⱼ SORTIE : eⱼ · yⱼ · (1−yⱼ) δⱼ CACHÉ : yⱼ·(1−yⱼ) · Σₖ δₖ·wₖⱼ Δwⱼᵢ = η · δⱼ · yᵢ + momentum : Δwⱼᵢ(n) = η·δⱼ·yᵢ + μ·Δwⱼᵢ(n-1) η = learning rate μ = momentum (inertie)
Vanishing gradient : sigmoïde/tanh → gradient ≈ 0 → couches profondes n'apprennent plus. Fix : ReLU.
Overfitting = mémorise → train ✓, test ✗
Fix : Dropout, L1/L2, Early stopping
CH 5

CNN — Réseaux Convolutifs

Architecture : Input → [Conv → ReLU → Pool]×N → Flatten → FC → Softmax
ReLU OBLIGATOIRE après chaque Conv. Pooling ne change PAS la profondeur D.
Conv : W_out = (W_in − F + 2P)/S + 1 Pool : W_out = (W − F)/S + 1 Params conv = F×F×D_in×K + K Exemple : 32×32×3, K=64, F=3, S=1, P=1 → W_out = (32−3+2)/1+1 = 32 → 32×32×64 MaxPool F=2 S=2 → 16×16×64
Pooling : ne change jamais D. Exemple : 32×32×64 après MaxPool(2,2) → 16×16×64 (pas 16×16×32 !)

CH 6

RNN · LSTM · Hopfield · Kohonen

Hopfield (1982)

  • Réseau récurrent entièrement connecté, poids symétriques
  • Neurones binaires (−1/+1)
  • Basé sur une fonction d'énergie
  • CAM : récupère un pattern depuis version bruitée/incomplète

LSTM (1997) — Hochreiter & Schmidhuber

fₜ = σ(Wf·[hₜ₋₁,xₜ]+bf) ← FORGET iₜ = σ(Wi·[hₜ₋₁,xₜ]+bi) ← INPUT oₜ = σ(Wo·[hₜ₋₁,xₜ]+bo) ← OUTPUT cₜ = fₜ⊙cₜ₋₁ + iₜ⊙c̃ₜ ← CELL hₜ = oₜ⊙tanh(cₜ)
fₜ≈0 = oublie cₜ₋₁ · iₜ≈1 = mémorise · résout vanishing gradient

Kohonen (SOM)

  • Non supervisé, grille 2D
  • BMU = neurone le plus proche de x
  • Met à jour BMU + voisins
  • Préserve la topologie des données
Elman (1990) : 1er RNN entraîné par backprop. Couche contexte = hₜ₋₁ réinjecté. Limite : vanishing gradient sur longues séquences.

Formules Complètes — À Maîtriser

Accuracy = (TP+TN) / (TP+TN+FP+FN) Precision = TP / (TP+FP) Recall = TP / (TP+FN) F1 Score = 2 × (P×R) / (P+R) Spécif. = TN / (TN+FP) MSE = (1/n) Σ(yᵢ − ŷᵢ)² R² = 1 − SS_res/SS_tot Marge SVM = 2/‖w‖ σ'(x) = σ(x)·(1−σ(x)) tanh'(x) = 1 − tanh²(x)
FP = Type I (fausse alarme) · FN = Type II (détection manquée)
Recall critique en médecine : FN = patient non traité → fatal
F1 Score : quand classes déséquilibrées. Ex : 99% sains → Acc=99% mais modèle nul !
Loss Function = erreur sur 1 exemple
Cost Function = moyenne sur tout le dataset
J(w) = (1/m)·Σ L(ŷ⁽ⁱ⁾, y⁽ⁱ⁾)

Q01 · CH1 ● Difficile
Dans la rétropropagation, le gradient local δⱼ pour une couche cachée est :
✅ B. Pour la couche de SORTIE : δ = e·y·(1−y) [réponse A]. Pour une couche CACHÉE : l'erreur n'est pas observable → on propage les δₖ de la couche suivante pondérés par wₖⱼ.
Q02 · CH3 ● Difficile
Image 64×64×3. Conv K=32, F=3, S=1, P=1. MaxPool F=2, S=2. Quelle est la taille finale ?
✅ B. Conv P=1: (64−3+2)/1+1 = 64 → 64×64×32. MaxPool: (64−2)/2+1 = 32 → 32×32×32. Le pooling divise W et H par 2, ne change pas K=32.
Q03 · Métriques ● Difficile
TP=80, FP=20, FN=40, TN=160. Quel est le F1-Score ?
✅ B. P = 80/100 = 0.80. R = 80/120 = 0.667. F1 = 2×0.80×0.667/(0.80+0.667) = 1.067/1.467 ≈ 0.727. Accuracy = 240/300 = 0.80.
Q04 · CH2 ● Moyen
Bayes : P(w1)=0.4, P(w2)=0.6, P(x/w1)=0.5, P(x/w2)=0.3. Décision pour x ?
✅ A. Comparer P(x/wᵢ)×P(wᵢ) : w1=0.5×0.4=0.20 vs w2=0.3×0.6=0.18. 0.20 > 0.18 → Décide w1. P(x) inutile à calculer.
Q05 · LSTM ● Difficile
Dans un LSTM, fₜ ≈ 0 signifie :
✅ C. cₜ = fₜ⊙cₜ₋₁ + iₜ⊙c̃ₜ. fₜ≈0 → fₜ⊙cₜ₋₁≈0 → la mémoire passée est effacée. Seule la nouvelle info iₜ⊙c̃ₜ reste.
Q06 · SVM ● Difficile
SVM avec εᵢ > 1 pour un point xᵢ. Ce point est :
✅ C. εᵢ=0 → bien classé hors marge. 0<εᵢ≤1 → bien classé mais dans la marge (VS). εᵢ>1 → MAL CLASSÉ, contrainte yᵢ(⟨w,xᵢ⟩+b) ≥ 1−εᵢ < 0.
Q07 · CNN ● Difficile
Conv(K=64, F=3, P=0, S=1) sur 32×32×3. Nombre de paramètres de cette couche ?
✅ B. Params = F×F×D_in×K + K (biais) = 3×3×3×64 + 64 = 1728 + 64 = 1792. Le "+K" = un biais par filtre.
Q08 · TPs ● Moyen
Pourquoi fit SEULEMENT sur X_train et transform sur X_test pour StandardScaler ?
✅ B. Si on fit sur le test, on utilise μ et σ du test pour normaliser → les paramètres du modèle sont influencés par des données qu'il ne devrait pas voir = data leakage = évaluation optimiste et fausse.
Q09 · CH1 ● Difficile
Différence fondamentale entre Eigenfaces (ACP) et Fisherfaces (LDA) ?
✅ C. ACP = compression optimale, pas de notion de classe. LDA = séparation maximale entre classes, robuste à l'illumination. Petite base → ACP peut dépasser LDA. Grande base → LDA meilleure.
Q10 · TPs ● Moyen
Dans le TP CNN COVID, pourquoi le Recall de la classe COVID est la métrique la plus critique ?
✅ B. Recall = TP/(TP+FN). Un FN = patient COVID classé "sain" → rentre chez lui, risque décès, contamination. Un FP = patient sain classé "COVID" → test supplémentaire, moins grave.
Q11 · MLP ● Difficile
L1 vs L2 regularisation. Laquelle met des poids EXACTEMENT à 0 ?
✅ A. L1 pénalité = Σ|wᵢ| → peut forcer des poids à exactement 0 (sélection de features). L2 pénalité = Σwᵢ² → réduit les poids proches de 0 mais jamais exactement 0.
Q12 · TPs ● Moyen
Loss function pour classification multi-classes Keras avec labels one-hot ?
✅ C. Labels one-hot [0,1,0] + softmax → categorical_crossentropy. Labels entiers (0,1,2) + softmax → sparse_categorical_crossentropy. Binary = 1 seul neurone sortie + sigmoïde.
Q13 · CH2 ● Difficile
CAH : quelle information encode la HAUTEUR d'un nœud dans le dendrogramme ?
✅ B. Plus la fusion est haute dans l'arbre, plus les clusters fusionnés sont éloignés. Couper le dendrogramme à une hauteur h donne des clusters dont la distance interne < h.
Q14 · Général ● Moyen
Différence entre Loss Function et Cost Function ?
✅ B. L(ŷ⁽ⁱ⁾, y⁽ⁱ⁾) = erreur sur un seul exemple. J(w) = (1/m)·Σ L(ŷ⁽ⁱ⁾, y⁽ⁱ⁾) = moyenne sur m exemples = Cost Function qu'on minimise pendant l'entraînement.
Q15 · CNN ● Difficile
Quel est le rôle du batch_size dans l'entraînement d'un CNN ?
✅ C. batch_size=32 → 32 images traitées → 1 mise à jour des poids. Petit batch → updates fréquentes mais bruyantes. Grand batch → updates stables. Avec 1000 images, 1 epoch = 1000/32 ≈ 31 updates.

Aide-mémoire — Tout d'un coup d'œil

Distances

Euclidienne: √Σ(aₖ-bₖ)² City Block : Σ|aₖ-bₖ| Mahalanobis: √((x-μ)ᵀΣ⁻¹(x-μ))

Bayes

P(wᵢ/x)=P(x/wᵢ)P(wᵢ)/P(x) P(x) = Σⱼ P(x/wⱼ)·P(wⱼ) Décide: max P(x/wᵢ)·P(wᵢ)

Confusion

Acc = (TP+TN)/total Pre = TP/(TP+FP) Rec = TP/(TP+FN) F1 = 2PR/(P+R)

SVM

Marge = 2/‖w‖ min ½‖w‖² Soft: +C·Σεᵢ RBF: exp(-‖x-y‖²/2σ²)

Backprop

δ sortie=e·y·(1-y) δ caché=y·(1-y)·Σδₖwₖⱼ Δw=η·δ·y +momentum: +μ·Δw(n-1)

CNN Dim.

Conv: (W-F+2P)/S+1 Pool: (W-F)/S+1 Params=F²·D·K+K

LSTM

fₜ=σ(Wf·[h,x]+b) FORGET iₜ=σ(Wi·[h,x]+b) INPUT oₜ=σ(Wo·[h,x]+b) OUTPUT cₜ=fₜ⊙cₜ₋₁+iₜ⊙c̃ₜ hₜ=oₜ⊙tanh(cₜ)

Activations

σ(x)=1/(1+e⁻ˣ) →(0,1) σ'(x)=σ(x)·(1-σ(x)) ReLU=max(0,x) Softmax=eˣⁱ/Σeˣʲ
⚠ Pièges classiques à l'examen : (1) Pooling ne change pas D. (2) δ couche cachée ≠ δ couche sortie. (3) FN = Type II (pas Type I). (4) ACP = non supervisé. (5) Soft margin : εᵢ>1 = mal classé. (6) OvO = N(N-1)/2, OvA = N.
