3. TP1 – TFD et Filtrage de Base

Objectif : analyser le signal, concevoir un filtre (PB ou PH), étudier ses réponses.

Étapes clés :
1
Lire et écouter le signal
2
Tracer signal temporel + spectre (dB)
3
Définir filtre (b, a)
4
Tracer pôles/zéros (zplane)
5
Réponse impulsionnelle h[n]
6
Réponse fréquentielle H(f)
7
Filtrer le signal et écouter
Passe-bas (PB)
H(z) = (z² + 2z + 1) / (z² − √2/2·z + 1/2)
b = [1, 2, 1]
a = [1, -0.5√2, 0.5]
Passe-haut (PH)
H(z) = (z² − 2z + 1) / (z² − √2/2·z + 1/2)
b = [1, -2, 1]
a = [1, -0.5√2, 0.5]
Différence : signe du coefficient central de b (+2 pour PB, −2 pour PH)

2.3 Vecteur temps
Te = 1/fe
N = len(x)
t = np.arange(N) * Te
t = [0, Te, 2Te, ..., (N-1)Te]
2.4 TFD et spectre PRIORITAIRE
NF = N
TFx = np.fft.fft(x, NF) / N
TFx = np.fft.fftshift(TFx)
TFxm = np.abs(TFx)
TFxdB = 20 * np.log10(TFxm)
f = np.arange(-NF/2, NF/2) * fe / NF
  • ÷ N : normalise (amplitude correcte)
  • fft → TFD complexe
  • fftshift → centre entre -fe/2 et +fe/2
  • abs → amplitude
  • 20log10 → passage en dB
  • f : axe des fréquences centré
Amplitude (dB)

2.6 Formatage audio (normalisation)
# x en uint8 [0..255]
n8 = 2**8
yf = np.uint8((y-min(y))*(n8-1) / (max(y)-min(y)))

# x en int16 [-32768..32767]
n16 = 2**16
yf = np.int16((y-min(y))*(n16-1) / (max(y)-min(y)) - n16//2)

# x en int8 [-128..127]

2.7 Filtrage
y = sp.filtfilt(b, a, x)
# b : numérateur H(z)
# a : dénominateur H(z)
# x : entrée
# y : sortie filtrée
RIF → a = 1 (pas de feedback)
2.8 Réponse impulsionnelle
Lh = 50
Dirac = np.zeros(Lh)
Dirac[0] = 1
h = sp.filtfilt(b, a, Dirac)
Dirac = δ[n] = 1 en n=0, 0 ailleurs
2.9 Réponse fréquentielle
L = 256
freq, Hf = sp.freqz(b, a, L, fs=fe)
|Hf| = np.abs(Hf)
|Hf|dB = 20*np.log10(np.abs(Hf)+5e-10)
# freq : de 0 à fe/2 (Hz)
2.10 Pôles et zéros
z, p = zplane(b, a)
poles = np.roots(a)
print(np.abs(poles))
⚠️ tous doivent être < 1
6. TP4 – Filtrage Multicadence
6.1 Décimation (sous-échantillonnage)
Sans filtre : x_d2 = x[::D]   (D = 2)
x_d2 = sp.decimate(x, D)
sp.decimate → filtre anti-repliement
puis décimation : fe_c = fe/(2D)
Écoute :
snd.play(x_d2, fe)
→ son accéléré !
snd.play(x_d2, fe/D)
→ son normal
Sans filtre :
snd.play(x_dz, fe)
→ son accéléré !
snd.play(x_dz, fe/D)
→ son normal
6.2 Interpolation (sur-échantillonnage)
Sans filtre :
  x_up = np.zeros(len(x)*D)
  x_up[::D] = x
Avec filtre : x_up_f = sp.resample(x, len(x)*D)
              sp.resample → rééchantillonnage
                            avec filtrage (FFT)
Écoute :
snd.play(x_up, fe)
→ son ralenti !
snd.play(x_up_f, fe*D)
→ son normal
Sans filtre :
snd.play(x_up, fe)
→ son ralenti !
snd.play(x_up_f, fe*D)
→ son normal
6.3 Changement rationnel (ex : 1.5 × fe)
Méthode 1 (directe)
x_res = sp.resample(x,
  int(len(x)*1.5))
Méthode 2 (3/2)
x_up3 = sp.resample(x,
  len(x)*3)
x_res = x_up3[::2]  # ÷2
⚠️ Règle d'or : TOUJOURS filtrer avant de décimer (sinon repliement) !
4. TP2 – Filtres RIF (Méthode des Fenêtres)

Objectif : débruiter avec un filtre RIF passe-bas (fenêtre). Étapes : lire → bruit → signal bruité → filtre RIF → filtrer

4.1 Génération du bruit
A = 50
bruit = A*(sin(2π*1500t)+sin(2π*2000t)
         +sin(2π*3000t))
# 3 sinusoïdes : 1500, 2000, 3000 Hz (ampl. = 50)
4.2 Ordre du filtre
deltaf = fe / 1000
Ncoef = int(3.3 / fe / deltaf)  # Hamming
if Ncoef % 2 == 0: Ncoef += 1
Rendre impair !
4.3 Conception RIF
fc = 1440 Hz
fe = 1500 Hz
b = firwin(Ncoef, fc/(fe/2), window='hamming')
f_c = fc/(fe/2)  # normalise entre 0 et 1
b = sp.firwin(Ncoef, f_c, window='hamming')
# Appliquer :  a = 1

5. TP3 – Filtres RII (Méthode Bilinéaire)

Objectif : débruiter avec des filtres RII (Butterworth, Cheby 1/II, Elliptique)

5.1 Paramètres
fc = 1400 Hz
delta_f = fe/50
fp = fc - delta_f/2
fa = fc + delta_f/2
Ap = 3 dB (ondulation BP)
Aa = 50 dB (att. BA)
fc = (fp + fa) / 2
5.2 Instructions (Python)
# Butterworth :
N,fc = sp.buttord(fp,fa,Ap,Aa,fs=fe)
b, a = sp.butter(N,fc,'low',fs=fe)
# Chebychev I :
N,fc = sp.cheb1ord(fp,fa,Ap,Aa,fs=fe)
b, a = sp.cheby1(N,Ap,fc,'low',fs=fe)
# Chebychev II :
N,fc = sp.cheb2ord(fp,fa,Ap,Aa,fs=fe)
b, a = sp.cheby2(N,Aa,fc,'low',fs=fe)
# Elliptique (Cauer) :
N,fc = sp.ellipord(N,Aa,fc,'low',fs=fe)
b, a = sp.ellip(N,Ap,Aa,fc,'low',fs=fe)
5.3 Comparatif des filtres RII

7. Que fait ce Code ? (Analyse rapide)
  • FFT + FFTSHIFT → spectre centré en fréq.
  • /N → normalisation (amplitude correcte)
  • Ncoef calcul → ordre filtre RIF Hamming
  • firwin(...) → filtre RIF passe-bas
  • sp.zeros(a), x[::D]=x → interpolation sans filtre
  • sp.decimate → décimation avec filtre
  • buttord(...) → calcule N et fc automatiquement
  • sp.applique le filtre (b,a)
  • lfilter(b, a, x) → filtre le signal
  • normalisation → pour écouter correctement
Formules utiles
Te = 1/fe
fc = fe / (fe/fc)
fc = (fp + fa) / 2
Δf = fa − fc
Mémo Mnémo
  • RIF → a = 1 (pas de retour)
  • RII → a ≠ 1 (retour présent)
  • Décimer → ↓ (moins d'éch.)
  • Interpoler → ↑ (plus d'éch.)
  • FFT + fftshift → spectre centré
